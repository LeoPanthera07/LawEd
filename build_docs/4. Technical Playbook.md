### 1.2 Rationale for Key Choices

**FastAPI over Django/Flask:** FastAPI is async-native, which is critical for a pipeline that makes parallel calls to Qdrant, Neo4j, and the LLM simultaneously. It generates OpenAPI docs automatically — useful for law firm API integration in Phase 2. Django is synchronous by default and carries significant overhead. Flask lacks built-in validation.

**LangGraph over raw LangChain or custom orchestration:** LangGraph models the pipeline as a stateful graph with typed nodes and conditional edges — exactly matching our architecture (proceed vs escalate routing, writeup trigger). It has built-in human-in-the-loop support via interrupt nodes, which is non-negotiable for our escalation path. Raw orchestration would require rebuilding this from scratch.

**Qdrant over pgvector or Pinecone:** pgvector is adequate for <1M vectors but fights PostgreSQL's connection model under heavy concurrent vector search. Pinecone is paid from day one. Qdrant is purpose-built, open-source, supports payload filtering (filter by act name, chapter), has a free self-hosted tier on Railway, and scales to 100M+ vectors without schema changes.

**Neo4j Community over Amazon Neptune or ArangoDB:** Neo4j Community is free, mature, has the best Python driver, and the APOC plugin provides the path traversal functions we need for GraphRAG. Neptune is AWS-locked and expensive. ArangoDB is less mature for graph-specific workloads.

**multilingual-e5-large over OpenAI ada-002 or IndicBERT:** multilingual-e5-large is open-source (runs locally, zero API cost), trained on 100 languages including Hindi, supports Hinglish reasonably well, and produces 1024-dimension vectors with strong semantic retrieval quality. OpenAI ada-002 costs per embedding and introduces an external dependency for a core function. IndicBERT is Hindi-focused but underperforms on English-Hindi mixed text.

**Groq + Llama 3.3 70B for MVP:** Groq's free tier provides 14,400 requests/day at ~300 tokens/second — more than sufficient for MVP query volumes. Llama 3.3 70B performs within 5% of GPT-4 on structured reasoning tasks. The LLM client abstraction means switching to Claude API in production requires changing one environment variable.

**Railway over AWS/GCP for MVP:** Railway eliminates Kubernetes, VPC, IAM, and load balancer configuration overhead — critical for a solo builder. It supports Docker-native deployment, managed PostgreSQL, internal service networking, and costs ~$0 on free tier for MVP volumes. The migration path to AWS/GCP is clean — the same Docker containers run anywhere.

## 2. Microservice Architecture

### 2.1 Architecture Decision — Modular Monolith First

For MVP, NyayaSetu is built as a **modular monolith** — a single FastAPI application with well-defined internal module boundaries. This is a deliberate architectural choice, not a shortcut.

Reasons:

- Solo builder — operational complexity of 5 microservices is not justified at MVP stage
- All modules share the same LangGraph pipeline state — splitting into services adds serialization overhead with no benefit
- Railway free tier makes running 5 separate services expensive — one service is free
- Module boundaries are enforced by folder structure and import rules — splitting into true microservices later requires moving folders, not redesigning

The system is designed so that any module can be extracted into an independent service when scale demands it. The internal API contracts (service layer interfaces) are identical to what REST APIs between microservices would look like.

## 3. Service Communication & API Design

### 3.1 Communication Patterns

**Internal (module-to-module):** Direct Python function calls — no serialization overhead. Each module exposes a service class with async methods.

**External (frontend-to-backend):** REST over HTTPS. All responses follow the standard envelope schema. Versioned under `/api/v1/`.

**External (backend-to-LLM):** HTTPS via Groq/Anthropic SDK. Async with retry logic.

**External (backend-to-Qdrant):** Qdrant Python client over HTTP — async.

**External (backend-to-Neo4j):** Neo4j async Python driver over Bolt protocol.

**External (Razorpay webhook):** POST to `/api/v1/payments/webhook` — signature verified before processing.

### 3.2 Standard Response Envelope

Every API response — success or failure — uses this envelope:

```python
# schemas/base.py
from pydantic import BaseModel
from typing import Generic, TypeVar, Optional
import uuid
from datetime import datetime

T = TypeVar("T")

class Meta(BaseModel):
    request_id: str = str(uuid.uuid4())
    timestamp: str = datetime.utcnow().isoformat() + "Z"
    version: str = "1.0"

class APIResponse(BaseModel, Generic[T]):
    success: bool
    data: Optional[T] = None
    error: Optional[dict] = None
    meta: Meta = Meta()

class ErrorDetail(BaseModel):
    code: str          # Machine-readable error code e.g. FREE_TIER_EXHAUSTED
    message: str       # Human-readable message
    field: Optional[str] = None  # For validation errors
```

### 3.3 HTTP Status Code Standards

|Situation|Status Code|
|---|---|
|Query accepted, processing async|202 Accepted|
|Resource created|201 Created|
|Success with data|200 OK|
|No content|204 No Content|
|Validation error|422 Unprocessable Entity|
|Authentication required|401 Unauthorized|
|Insufficient permissions|403 Forbidden|
|Resource not found|404 Not Found|
|Payment required|402 Payment Required|
|Rate limit exceeded|429 Too Many Requests|
|Conflict (duplicate, replay attack)|409 Conflict|
|Internal server error|500 Internal Server Error|
|Service unavailable|503 Service Unavailable|

## 4. Database Design & Schema

### 4.1 PostgreSQL — Complete Schema

```sql
-- Enable UUID extension
CREATE EXTENSION IF NOT EXISTS "pgcrypto";

-- ═══════════════════════════════════════════════════════
-- USERS
-- ═══════════════════════════════════════════════════════
CREATE TABLE users (
    id                      UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email                   VARCHAR(255) UNIQUE NOT NULL,
    phone                   VARCHAR(20),
    full_name               VARCHAR(255) NOT NULL,
    password_hash           VARCHAR(255) NOT NULL,
    role                    VARCHAR(50) NOT NULL DEFAULT 'citizen'
                            CHECK (role IN ('citizen', 'lawyer', 'admin')),
    is_active               BOOLEAN NOT NULL DEFAULT TRUE,
    is_verified             BOOLEAN NOT NULL DEFAULT FALSE,
    email_verification_token VARCHAR(255),
    email_verification_expires_at TIMESTAMPTZ,
    password_reset_token    VARCHAR(255),
    password_reset_expires_at TIMESTAMPTZ,
    bar_council_id          VARCHAR(100),
    free_queries_used       INTEGER NOT NULL DEFAULT 0,
    free_queries_reset_at   TIMESTAMPTZ NOT NULL
                            DEFAULT (NOW() + INTERVAL '30 days'),
    last_login_at           TIMESTAMPTZ,
    created_at              TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at              TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_users_email ON users(email);
CREATE INDEX idx_users_role ON users(role);
CREATE INDEX idx_users_is_active ON users(is_active);

-- ═══════════════════════════════════════════════════════
-- SESSIONS (refresh token tracking)
-- ═══════════════════════════════════════════════════════
CREATE TABLE sessions (
    id              UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id         UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    refresh_token   VARCHAR(512) NOT NULL UNIQUE,
    is_revoked      BOOLEAN NOT NULL DEFAULT FALSE,
    user_agent      VARCHAR(500),
    ip_address      INET,
    expires_at      TIMESTAMPTZ NOT NULL,
    created_at      TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    last_used_at    TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_sessions_user_id ON sessions(user_id);
CREATE INDEX idx_sessions_refresh_token ON sessions(refresh_token);
CREATE INDEX idx_sessions_expires_at ON sessions(expires_at);

-- ═══════════════════════════════════════════════════════
-- QUERIES
-- ═══════════════════════════════════════════════════════
CREATE TABLE queries (
    id                  UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id             UUID REFERENCES users(id) ON DELETE SET NULL,
    input_text          TEXT NOT NULL,
    input_language      VARCHAR(20),
    translated_text     TEXT,
    status              VARCHAR(50) NOT NULL DEFAULT 'pending'
                        CHECK (status IN (
                            'pending', 'processing', 'completed',
                            'escalated', 'failed'
                        )),
    pipeline_run_id     VARCHAR(255),
    pipeline_error      TEXT,
    is_guest            BOOLEAN NOT NULL DEFAULT FALSE,
    ip_address          INET,
    created_at          TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    processing_started_at TIMESTAMPTZ,
    completed_at        TIMESTAMPTZ,
    processing_duration_ms INTEGER
);

CREATE INDEX idx_queries_user_id ON queries(user_id);
CREATE INDEX idx_queries_status ON queries(status);
CREATE INDEX idx_queries_created_at ON queries(created_at DESC);
CREATE INDEX idx_queries_user_status ON queries(user_id, status);

-- ═══════════════════════════════════════════════════════
-- CLAUSE MAPS (pipeline output)
-- ═══════════════════════════════════════════════════════
CREATE TABLE clause_maps (
    id                  UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    query_id            UUID NOT NULL UNIQUE REFERENCES queries(id)
                        ON DELETE CASCADE,
    clauses             JSONB NOT NULL DEFAULT '[]',
    confidence_score    DECIMAL(5,4),
    acts_covered        TEXT[] DEFAULT '{}',
    clause_count        INTEGER NOT NULL DEFAULT 0,
    is_lawyer_reviewed  BOOLEAN NOT NULL DEFAULT FALSE,
    lawyer_reviewed_by  UUID REFERENCES users(id),
    lawyer_reviewed_at  TIMESTAMPTZ,
    created_at          TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_at          TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_clause_maps_query_id ON clause_maps(query_id);
CREATE INDEX idx_clause_maps_confidence ON clause_maps(confidence_score);
CREATE INDEX idx_clause_maps_clauses ON clause_maps USING gin(clauses);

-- ═══════════════════════════════════════════════════════
-- ESCALATIONS
-- ═══════════════════════════════════════════════════════
CREATE TABLE escalations (
    id                  UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    query_id            UUID NOT NULL REFERENCES queries(id),
    clause_map_id       UUID REFERENCES clause_maps(id),
    reason              VARCHAR(50) NOT NULL
                        CHECK (reason IN (
                            'low_confidence', 'high_stakes', 'manual'
                        )),
    confidence_score    DECIMAL(5,4),
    high_stakes_flag    VARCHAR(100),
    status              VARCHAR(50) NOT NULL DEFAULT 'pending'
                        CHECK (status IN (
                            'pending', 'under_review', 'resolved', 'flagged'
                        )),
    assigned_lawyer_id  UUID REFERENCES users(id),
    assigned_at         TIMESTAMPTZ,
    lawyer_notes        TEXT,
    resolution          VARCHAR(50)
                        CHECK (resolution IN ('approved', 'edited', 'flagged')),
    approved_for_writeup BOOLEAN NOT NULL DEFAULT FALSE,
    resolved_at         TIMESTAMPTZ,
    citizen_notified_at TIMESTAMPTZ,
    created_at          TIMESTAMPTZ NOT NULL DEFAULT NOW()
);

CREATE INDEX idx_escalations_status ON escalations(status);
CREATE INDEX idx_escalations_assigned_lawyer ON escalations(assigned_lawyer_id);
CREATE INDEX idx_escalations_query_id ON escalations(query_id);
CREATE INDEX idx_escalations_created_at ON escalations(created_at DESC);

-- ═══════════════════════════════════════════════════════
-- PAYMENTS
-- ═══════════════════════════════════════════════════════
CREATE TABLE payments (
    id                      UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id                 UUID NOT NULL REFERENCES users(id),
    query_id                UUID REFERENCES queries(id),
    razorpay_order_id       VARCHAR(100) UNIQUE NOT NULL,
    razorpay_payment_id     VARCHAR(100) UNIQUE,
```

```python
# data_pipeline/pdf_ingestion.py

import fitz  # PyMuPDF
import json
import re
from pathlib import Path
from dataclasses import dataclass
from typing import List, Optional

@dataclass
class ClauseChunk:
    act: str
    chapter: str
    chapter_number: str
    section_number: str
    section_title: str
    clause_text: str
    chunk_index: int
    total_chunks: int
    word_count: int

class LegalPDFIngester:
    """
    Ingests BNS, BNSS, BSA PDFs into structured clause chunks.

    PDF structure assumption:
    - Chapter heading: "CHAPTER {roman_numeral}"
    - Section heading: "{number}. {Title}."
    - Subsections: "({a})", "({i})", etc.
    """

    MAX_CHUNK_TOKENS = 512
    CHUNK_OVERLAP_TOKENS = 50

    def __init__(self, act_name: str):
        self.act_name = act_name  # "BNS" | "BNSS" | "BSA"
        self.chunks: List[ClauseChunk] = []

    def ingest(self, pdf_path: str) -> List[ClauseChunk]:
        doc = fitz.open(pdf_path)
        full_text = ""
        for page in doc:
            full_text += page.get_text("text")
        doc.close()

        # Parse into structured sections
        sections = self._parse_sections(full_text)
        # Chunk long sections
        for section in sections:
            self._chunk_section(section)

        return self.chunks

    def _parse_sections(self, text: str) -> List[dict]:
        """
        Extract sections using regex patterns for Indian legal document structure.
        Returns list of {chapter, section_number, section_title, full_text}
        """
        sections = []
        current_chapter = ""
        current_chapter_num = ""

        # Chapter pattern: "CHAPTER I" / "CHAPTER XIV"
        chapter_pattern = re.compile(
            r'CHAPTER\s+([IVXLCDM]+)\s*\n\s*([A-Z][^\n]+)',
            re.MULTILINE
        )
        # Section pattern: "1. Short title." / "103. Punishment for murder."
        section_pattern = re.compile(
            r'^(\d+[A-Z]?)\.\s+([^.\n]+\.)\s*\n(.*?)(?=^\d+[A-Z]?\.|^CHAPTER|\Z)',
            re.MULTILINE | re.DOTALL
        )

        for chapter_match in chapter_pattern.finditer(text):
            current_chapter_num = chapter_match.group(1)
            current_chapter = chapter_match.group(2).strip()

        for section_match in section_pattern.finditer(text):
            sections.append({
                "chapter": current_chapter,
                "chapter_number": current_chapter_num,
                "section_number": section_match.group(1),
                "section_title": section_match.group(2).strip(),
                "full_text": section_match.group(3).strip()
            })

        return sections

    def _chunk_section(self, section: dict) -> None:
        text = section["full_text"]
        words = text.split()

        if len(words) <= self.MAX_CHUNK_TOKENS:
            # Single chunk
            self.chunks.append(ClauseChunk(
                act=self.act_name,
                chapter=section["chapter"],
                chapter_number=section["chapter_number"],
                section_number=section["section_number"],
                section_title=section["section_title"],
                clause_text=f"{section['section_number']}. {section['section_title']} {text}",
                chunk_index=0,
                total_chunks=1,
                word_count=len(words)
            ))
        else:
            # Split into overlapping chunks
            chunks = []
            step = self.MAX_CHUNK_TOKENS - self.CHUNK_OVERLAP_TOKENS
            for i in range(0, len(words), step):
                chunk_words = words[i:i + self.MAX_CHUNK_TOKENS]
                chunk_text = " ".join(chunk_words)
                # Prepend section title to every chunk for context
                prefixed = f"{section['section_number']}. {section['section_title']} {chunk_text}"
                chunks.append(prefixed)

            for idx, chunk_text in enumerate(chunks):
                self.chunks.append(ClauseChunk(
                    act=self.act_name,
                    chapter=section["chapter"],
                    chapter_number=section["chapter_number"],
                    section_number=section["section_number"],
                    section_title=section["section_title"],
                    clause_text=chunk_text,
                    chunk_index=idx,
                    total_chunks=len(chunks),
                    word_count=len(chunk_text.split())
                ))
```

```python
# data_pipeline/embedder.py

from sentence_transformers import SentenceTransformer
from qdrant_client import QdrantClient
from qdrant_client.models import PointStruct
import uuid
import numpy as np
from typing import List
import logging

logger = logging.getLogger(__name__)

class CorpusEmbedder:
    """
    Embeds clause chunks and upserts to Qdrant.
    Uses multilingual-e5-large with instruction prefix for retrieval.
    """

    MODEL_NAME = "intfloat/multilingual-e5-large"
    BATCH_SIZE = 32
    # e5 models require instruction prefix for passage encoding
    PASSAGE_PREFIX = "passage: "
    QUERY_PREFIX = "query: "

    def __init__(self):
        logger.info(f"Loading embedding model: {self.MODEL_NAME}")
        self.model = SentenceTransformer(self.MODEL_NAME)
        logger.info("Embedding model loaded")

    def embed_passages(self, texts: List[str]) -> np.ndarray:
        """Embed corpus passages with passage prefix"""
        prefixed = [f"{self.PASSAGE_PREFIX}{t}" for t in texts]
        return self.model.encode(
            prefixed,
            batch_size=self.BATCH_SIZE,
            show_progress_bar=True,
            normalize_embeddings=True  # cosine similarity requires normalized vectors
        )

    def embed_query(self, query: str) -> np.ndarray:
        """Embed a single query with query prefix"""
        prefixed = f"{self.QUERY_PREFIX}{query}"
        return self.model.encode(
            prefixed,
            normalize_embeddings=True
        )

    def upsert_to_qdrant(
        self,
        chunks: List[ClauseChunk],
        qdrant_client: QdrantClient,
        collection_name: str
    ) -> None:
        texts = [chunk.clause_text for chunk in chunks]
        embeddings = self.embed_passages(texts)

        points = []
        for chunk, embedding in zip(chunks, embeddings):
            points.append(PointStruct(
                id=str(uuid.uuid4()),
                vector=embedding.tolist(),
                payload={
                    "act": chunk.act,
                    "chapter": chunk.chapter,
                    "chapter_number": chunk.chapter_number,
                    "section_number": chunk.section_number,
                    "section_title": chunk.section_title,
                    "clause_text": chunk.clause_text,
                    "chunk_index": chunk.chunk_index,
                    "total_chunks": chunk.total_chunks,
                    "word_count": chunk.word_count,
                    "language": "en",
                    "version": "1.0"
                }
            ))

        # Upsert in batches of 100
        for i in range(0, len(points), 100):
            batch = points[i:i+100]
            qdrant_client.upsert(
                collection_name=collection_name,
                points=batch
            )
            logger.info(f"Upserted batch {i//100 + 1}: {len(batch)} vectors")
```

### 5.3 Graph Extraction Pipeline

```python
# data_pipeline/graph_extractor.py

RELATIONSHIP_EXTRACTION_PROMPT = """
You are a senior Indian legal expert analysing the relationships between clauses
in BNS (Bharatiya Nyaya Sanhita), BNSS (Bharatiya Nagarik Suraksha Sanhita),
and BSA (Bharatiya Sakshya Adhiniyam).

Analyse this clause and identify ALL relationships to other clauses:

Source clause: {act} Section {section_number} — {section_title}
Text: {clause_text}

Valid relationship types:
- REFERENCES: This clause explicitly mentions or cites another clause
- PROCEDURAL_DEPENDENCY: This clause requires a procedure defined in another clause
- EVIDENCE_REQUIREMENT: This clause requires evidence governed by another clause (usually BSA)
- CROSS_ACT_LINK: Semantic connection across the three acts
- EXCEPTION_OF: This clause is an exception or proviso to another clause
- AGGRAVATED_BY: Another clause describes aggravated punishment for this offence

For each relationship, you MUST verify the target section actually exists.
Only include relationships you are highly confident about (confidence > 0.7).

Return ONLY valid JSON in this exact format:
{{
  "relationships": [
    {{
      "target_act": "BNS" | "BNSS" | "BSA",
      "target_section": "<section number>",
      "relationship_type": "<one of the types above>",
      "confidence": <float 0.7-1.0>,
      "reasoning": "<one sentence explaining why>"
    }}
  ]
}}

If no confident relationships exist, return: {{"relationships": []}}
"""

class GraphExtractor:
    def __init__(self, llm_client, neo4j_client, valid_sections: set):
        self.llm = llm_client
        self.neo4j = neo4j_client
        # Pre-build set of all valid section IDs for hallucination guard
        self.valid_sections = valid_sections  # {"BNS-103", "BNSS-173", ...}

    async def extract_and_store(self, chunk: ClauseChunk) -> int:
        """Returns count of relationships stored"""
        prompt = RELATIONSHIP_EXTRACTION_PROMPT.format(
            act=chunk.act,
            section_number=chunk.section_number,
            section_title=chunk.section_title,
            clause_text=chunk.clause_text[:1000]  # Truncate for token budget
        )

        try:
            response = await self.llm.complete(prompt, max_tokens=500)
            data = json.loads(response)
            relationships = data.get("relationships", [])
        except Exception as e:
            logger.error(f"Graph extraction failed for {chunk.act}-{chunk.section_number}: {e}")
            return 0

        stored = 0
        for rel in relationships:
            target_id = f"{rel['target_act']}-{rel['target_section']}"

            # CRITICAL: Only store relationships to sections that actually exist
            if target_id not in self.valid_sections:
                logger.warning(f"Skipping invalid target: {target_id}")
                continue

            if rel["confidence"] < 0.7:
                continue

            await self._store_relationship(chunk, rel)
            stored += 1

        return stored

    async def _store_relationship(self, source: ClauseChunk, rel: dict):
        source_id = f"{source.act}-{source.section_number}"
        target_id = f"{rel['target_act']}-{rel['target_section']}"
        rel_type = rel["relationship_type"]

        cypher = f"""
        MATCH (source:Clause {{id: $source_id}})
        MATCH (target:Clause {{id: $target_id}})
        MERGE (source)-[r:{rel_type}]->(target)
        SET r.confidence = $confidence,
            r.reasoning = $reasoning,
            r.created_at = datetime()
        """
        await self.neo4j.run(
            cypher,
            source_id=source_id,
            target_id=target_id,
            confidence=rel["confidence"],
            reasoning=rel["reasoning"]
        )
```

```python
# pipeline/graph.py

from langgraph.graph import StateGraph, END
from langgraph.checkpoint.memory import MemorySaver
from pipeline.state import PipelineState, PipelineStatus
from pipeline.nodes import (
    input_validator_node,
    language_detector_node,
    query_expander_node,
    qdrant_retriever_node,
    neo4j_traverser_node,
    result_merger_node,
    confidence_evaluator_node,
    response_synthesiser_node,
    output_formatter_node,
    escalation_router_node
)

def build_pipeline() -> StateGraph:
    """
    Build the NyayaSetu LangGraph pipeline.

    Graph structure:
    entry → language_detector → query_expander → [qdrant_retriever + neo4j_traverser]
    → result_merger → confidence_evaluator → [proceed: response_synthesiser | escalate]
    → output_formatter → END
    """
    workflow = StateGraph(PipelineState)

    # Add nodes
    workflow.add_node("input_validator", input_validator_node)
    workflow.add_node("language_detector", language_detector_node)
    workflow.add_node("query_expander", query_expander_node)
    workflow.add_node("qdrant_retriever", qdrant_retriever_node)
    workflow.add_node("neo4j_traverser", neo4j_traverser_node)
    workflow.add_node("result_merger", result_merger_node)
    workflow.add_node("confidence_evaluator", confidence_evaluator_node)
    workflow.add_node("response_synthesiser", response_synthesiser_node)
    workflow.add_node("output_formatter", output_formatter_node)
    workflow.add_node("escalation_router", escalation_router_node)

    # Set entry point
    workflow.set_entry_point("input_validator")

    # Linear edges
    workflow.add_edge("input_validator", "language_detector")
    workflow.add_edge("language_detector", "query_expander")
    workflow.add_edge("query_expander", "qdrant_retriever")
    workflow.add_edge("qdrant_retriever", "neo4j_traverser")
    workflow.add_edge("neo4j_traverser", "result_merger")
    workflow.add_edge("result_merger", "confidence_evaluator")

    # Conditional edge — the core routing decision
    workflow.add_conditional_edges(
        "confidence_evaluator",
        route_after_confidence,
        {
            "proceed": "response_synthesiser",
            "escalate": "escalation_router"
        }
    )

    # Proceed path
    workflow.add_edge("response_synthesiser", "output_formatter")
    workflow.add_edge("output_formatter", END)

    # Escalation path
    workflow.add_edge("escalation_router", END)

    return workflow.compile()

def route_after_confidence(state: PipelineState) -> str:
    """
    Routing function for conditional edge.
    Returns "proceed" or "escalate".
    """
    return state.get("routing_decision", "escalate")

# Singleton pipeline instance — compiled once at startup
_pipeline = None

def get_pipeline() -> StateGraph:
    global _pipeline
    if _pipeline is None:
        _pipeline = build_pipeline()
    return _pipeline

async def run_pipeline(query_id: str, raw_input: str) -> dict:
    """
    Main entry point for running a query through the pipeline.
    Returns the final state dict.
    """
    pipeline = get_pipeline()

    initial_state = PipelineState(
        query_id=query_id,
        raw_input=raw_input,
        errors=[],
        status=PipelineStatus.PROCESSING
    )

    config = {"configurable": {"thread_id": query_id}}

    try:
        final_state = await pipeline.ainvoke(initial_state, config=config)
        return final_state
    except Exception as e:
        logger.error(f"Pipeline failed for query {query_id}: {e}", exc_info=True)
        return {
            "query_id": query_id,
            "status": PipelineStatus.FAILED,
            "errors": [str(e)],
            "final_output": None
        }
```

```python
# clients/llm_client.py

from abc import ABC, abstractmethod
from typing import Optional
import asyncio
import logging
from config import settings

logger = logging.getLogger(__name__)

class BaseLLMClient(ABC):
    MAX_RETRIES = 3
    RETRY_DELAYS = [2, 4, 8]  # Exponential backoff seconds

    @abstractmethod
    async def _complete_once(self, prompt: str, max_tokens: int) -> str:
        pass

    async def complete(
        self,
        prompt: str,
        max_tokens: int = 1000,
        temperature: float = 0.1
    ) -> str:
        last_error = None
        for attempt, delay in enumerate(self.RETRY_DELAYS):
            try:
                result = await self._complete_once(prompt, max_tokens)
                if attempt > 0:
                    logger.info(f"LLM succeeded on retry {attempt + 1}")
                return result
            except Exception as e:
                last_error = e
                error_type = type(e).__name__
                logger.warning(
                    f"LLM attempt {attempt + 1} failed: {error_type}: {e}"
                )
                if attempt < len(self.RETRY_DELAYS) - 1:
                    await asyncio.sleep(delay)

        raise RuntimeError(
            f"LLM failed after {self.MAX_RETRIES} retries: {last_error}"
        )


class GroqClient(BaseLLMClient):
    MODEL = "llama-3.3-70b-versatile"

    def __init__(self):
        from groq import AsyncGroq
        self.client = AsyncGroq(api_key=settings.GROQ_API_KEY)

    async def _complete_once(self, prompt: str, max_tokens: int) -> str:
        response = await self.client.chat.completions.create(
            model=self.MODEL,
            messages=[{"role": "user", "content": prompt}],
            max_tokens=max_tokens,
            temperature=0.1,
            timeout=30.0
        )
        return response.choices[0].message.content

        )

def get_llm_client() -> BaseLLMClient:
    """
    Factory function — LLM provider determined by environment variable.
    Swap from groq to claude by changing LLM_PROVIDER in Railway dashboard.
    """
    provider = settings.LLM_PROVIDER.lower()
    if provider == "groq":
        return GroqClient()
    elif provider == "claude":
        return ClaudeClient()
    elif provider == "mock":
        return MockLLMClient()
    else:
        raise ValueError(f"Unknown LLM provider: {provider}")
```

### 7.1 JWT Strategy

```python
# core/security.py

from datetime import datetime, timedelta
from typing import Optional
from jose import JWTError, jwt
from passlib.context import CryptContext
from config import settings

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto", bcrypt__rounds=12)

class TokenType:
    ACCESS = "access"
    REFRESH = "refresh"

def create_access_token(user_id: str, role: str) -> str:
    payload = {
        "sub": user_id,
        "role": role,
        "type": TokenType.ACCESS,
        "iat": datetime.utcnow(),
        "exp": datetime.utcnow() + timedelta(
            minutes=settings.JWT_ACCESS_TOKEN_EXPIRE_MINUTES
        )
    }
    return jwt.encode(payload, settings.APP_SECRET_KEY, algorithm="HS256")

def create_refresh_token(user_id: str) -> str:
    payload = {
        "sub": user_id,
        "type": TokenType.REFRESH,
        "iat": datetime.utcnow(),
        "exp": datetime.utcnow() + timedelta(
            days=settings.JWT_REFRESH_TOKEN_EXPIRE_DAYS
        )
    }
    return jwt.encode(payload, settings.APP_SECRET_KEY, algorithm="HS256")

def verify_token(token: str, expected_type: str) -> dict:
    try:
        payload = jwt.decode(token, settings.APP_SECRET_KEY, algorithms=["HS256"])
        if payload.get("type") != expected_type:
            raise JWTError("Wrong token type")
        return payload
    except JWTError as e:
        raise InvalidTokenError(str(e))

def hash_password(password: str) -> str:
    return pwd_context.hash(password)

def verify_password(plain: str, hashed: str) -> bool:
    return pwd_context.verify(plain, hashed)
```

### 7.2 RBAC Middleware

```python
# core/rbac.py

from functools import wraps
from fastapi import Depends, HTTPException, status
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from core.security import verify_token
from core.exceptions import InsufficientPermissionsError

security = HTTPBearer()

class RBACGuard:
    def __init__(self, allowed_roles: list[str]):
        self.allowed_roles = allowed_roles

    async def __call__(
        self,
        credentials: HTTPAuthorizationCredentials = Depends(security)
    ) -> dict:
        token = credentials.credentials
        try:
            payload = verify_token(token, "access")
        except Exception:
            raise HTTPException(
                status_code=status.HTTP_401_UNAUTHORIZED,
                detail={"code": "INVALID_TOKEN", "message": "Invalid or expired token"}
            )

        user_role = payload.get("role")
        if user_role not in self.allowed_roles:
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail={
                    "code": "INSUFFICIENT_PERMISSIONS",
                    "message": f"This action requires one of: {self.allowed_roles}"
                }
            )

        return payload  # Returns {"sub": user_id, "role": role, ...}

# Reusable dependency instances
require_citizen = RBACGuard(["citizen", "lawyer", "admin"])
require_lawyer = RBACGuard(["lawyer", "admin"])
require_admin = RBACGuard(["admin"])
```

### 9.2 Request Logging Middleware

```python
# main.py — request logging

import time
import structlog

logger = structlog.get_logger()

@app.middleware("http")
async def log_requests(request, call_next):
    request_id = str(uuid.uuid4())
    start_time = time.monotonic()

    structlog.contextvars.bind_contextvars(request_id=request_id)

    response = await call_next(request)

    duration_ms = (time.monotonic() - start_time) * 1000

    logger.info(
        "http_request",
        method=request.method,
        path=request.url.path,
        status_code=response.status_code,
        duration_ms=round(duration_ms, 2),
        client_ip=request.client.host if request.client else None
    )

    structlog.contextvars.clear_contextvars()
    response.headers["X-Request-ID"] = request_id
    return response
```

```python
            "pipeline_node_start",
            node=node_name,
            query_id=state["query_id"]
        )
        try:
            result = await node_fn(state)
            duration_ms = (time.monotonic() - start) * 1000
            logger.info(
                "pipeline_node_complete",
                node=node_name,
                query_id=state["query_id"],
                duration_ms=round(duration_ms, 2),
                errors_count=len(result.get("errors", []))
            )
            return result
        except Exception as e:
            duration_ms = (time.monotonic() - start) * 1000
            logger.error(
                "pipeline_node_failed",
                node=node_name,
                query_id=state["query_id"],
                duration_ms=round(duration_ms, 2),
                error=str(e),
                exc_info=True
            )
            raise
    return wrapper
```

### 10.3 Circuit Breaker — Neo4j

```python
# clients/neo4j_client.py

import asyncio
import time
from enum import Enum

class CircuitState(Enum):
    CLOSED = "closed"       # Normal operation
    OPEN = "open"           # Failing — reject all calls
    HALF_OPEN = "half_open" # Testing recovery

class CircuitBreaker:
    def __init__(
        self,
        failure_threshold: int = 3,
        recovery_timeout: int = 300,   # 5 minutes
        half_open_timeout: int = 30
    ):
        self.failure_threshold = failure_threshold
        self.recovery_timeout = recovery_timeout
        self.failure_count = 0
        self.last_failure_time = None
        self.state = CircuitState.CLOSED

    def record_failure(self):
        self.failure_count += 1
        self.last_failure_time = time.monotonic()
        if self.failure_count >= self.failure_threshold:
            self.state = CircuitState.OPEN
            logger.warning("circuit_breaker_opened", service="neo4j")

    def record_success(self):
        self.failure_count = 0
        self.state = CircuitState.CLOSED

    def can_attempt(self) -> bool:
        if self.state == CircuitState.CLOSED:
            return True
        if self.state == CircuitState.OPEN:
            if time.monotonic() - self.last_failure_time > self.recovery_timeout:
                self.state = CircuitState.HALF_OPEN
                return True
            return False
        return True  # HALF_OPEN — allow one attempt

neo4j_circuit_breaker = CircuitBreaker()

class Neo4jClientWrapper:
    async def run(self, query: str, **params):
        if not neo4j_circuit_breaker.can_attempt():
            raise Neo4jCircuitOpenError("Neo4j circuit breaker is open")

        try:
            result = await self._execute(query, **params)
            neo4j_circuit_breaker.record_success()
            return result
        except Exception as e:
            neo4j_circuit_breaker.record_failure()
            raise
```
