## 3. LangGraph Pipeline — Detailed Node Specifications

### 3.1 Pipeline State Schema

```python
# pipeline/state.py
from typing import TypedDict, Optional, List
from enum import Enum

class Language(str, Enum):
    ENGLISH = "en"
    HINDI = "hi"
    HINGLISH = "hinglish"

class PipelineStatus(str, Enum):
    PROCESSING = "processing"
    COMPLETED = "completed"
    ESCALATED = "escalated"
    FAILED = "failed"

class ClauseResult(TypedDict):
    rank: int
    act: str                    # 'BNS' | 'BNSS' | 'BSA'
    section_number: str
    section_title: str
    clause_text: str
    plain_explanation: str
    relevance_score: float
    relevance_context: str
    source: str                 # 'qdrant' | 'neo4j'

class PipelineState(TypedDict):
    # Input
    query_id: str
    raw_input: str

    # Node 1 output
    detected_language: Optional[Language]
    translated_input: Optional[str]
    detected_acts: Optional[List[str]]

    # Node 2 output
    expanded_queries: Optional[List[str]]

    # Node 3A output
    qdrant_results: Optional[List[ClauseResult]]

    # Node 3B output
    neo4j_results: Optional[List[ClauseResult]]

    # Node 4 output
    merged_clauses: Optional[List[ClauseResult]]

    # Node 5 output
    confidence_score: Optional[float]
    high_stakes_flag: Optional[str]
    routing_decision: Optional[str]    # 'proceed' | 'escalate'

    # Node 6 output
    synthesised_clauses: Optional[List[ClauseResult]]

    # Node 7 output
    final_output: Optional[dict]

    # Error tracking
    errors: List[str]
    status: PipelineStatus
```

### 3.2 Node 1 — Language and Intent Detector

```python
# pipeline/nodes/language_detector.py
from pipeline.state import PipelineState, Language
from clients.llm_client import LLMClient

LANGUAGE_DETECTION_PROMPT = """
You are a language detection and legal intent analysis system.

Given the following text from an Indian citizen:
<text>{input_text}</text>

Respond in JSON with exactly this structure:
{{
  "detected_language": "en" | "hi" | "hinglish",
  "english_translation": "<full English translation, or same text if already English>",
  "detected_acts": ["BNS", "BNSS", "BSA"],  // which acts likely apply - include all that may be relevant
  "incident_category": "<one of: assault, property, road_accident, workplace, cyber, domestic, fraud, other>",
  "high_stakes_indicators": ["<list any: life_at_risk, liberty_at_risk, child_involved, serious_bodily_harm>"]
}}

Rules:
- detected_acts should include BNS if a criminal offence is described
- detected_acts should include BNSS if procedure (FIR, arrest, bail) is relevant
- detected_acts should include BSA if evidence (CCTV, documents, witnesses) is mentioned
- If uncertain, include all three acts
- Never add commentary outside the JSON structure
"""

async def language_detector_node(state: PipelineState) -> PipelineState:
    llm = LLMClient()
    try:
        prompt = LANGUAGE_DETECTION_PROMPT.format(
            input_text=state["raw_input"]
        )
        response = await llm.complete(prompt, max_tokens=300)
        parsed = json.loads(response)

        state["detected_language"] = parsed["detected_language"]
        state["translated_input"] = parsed["english_translation"]
        state["detected_acts"] = parsed["detected_acts"]

        # Store high stakes indicators for Node 5
        state["_high_stakes_raw"] = parsed.get("high_stakes_indicators", [])

    except Exception as e:
        state["errors"].append(f"language_detector: {str(e)}")
        state["detected_language"] = Language.ENGLISH
        state["translated_input"] = state["raw_input"]
        state["detected_acts"] = ["BNS", "BNSS", "BSA"]

    return state
```

### 3.3 Node 2 — Query Expander

```python
# pipeline/nodes/query_expander.py

QUERY_EXPANSION_PROMPT = """
You are a legal query expansion system for Indian law.

Original citizen query (English): {translated_input}
Detected acts: {detected_acts}
Incident category: {incident_category}

Generate 4 specific retrieval queries that will find the most relevant clauses
in BNS (Bharatiya Nyaya Sanhita), BNSS (Bharatiya Nagarik Suraksha Sanhita),
and BSA (Bharatiya Sakshya Adhiniyam).

Each query should target a different aspect:
1. The primary criminal offence described
2. The procedural steps the citizen should take
3. The evidence requirements for this type of case
4. Any related offences or aggravating factors

Respond as JSON:
{{
  "queries": [
    "<query 1>",
    "<query 2>",
    "<query 3>",
    "<query 4>"
  ]
}}

Keep each query under 30 words. Use legal terminology where it helps precision.
"""

async def query_expander_node(state: PipelineState) -> PipelineState:
    llm = LLMClient()
    try:
        prompt = QUERY_EXPANSION_PROMPT.format(
            translated_input=state["translated_input"],
            detected_acts=state["detected_acts"],
            incident_category=state.get("_incident_category", "other")
        )
        response = await llm.complete(prompt, max_tokens=400)
        parsed = json.loads(response)
        state["expanded_queries"] = parsed["queries"]

    except Exception as e:
        state["errors"].append(f"query_expander: {str(e)}")
        # Fallback: use translated input as sole query
        state["expanded_queries"] = [state["translated_input"]]

    return state
```

### 3.4 Node 3A — Qdrant Retriever

```python
# pipeline/nodes/qdrant_retriever.py
from clients.qdrant_client import QdrantClientWrapper
from clients.embedding_client import EmbeddingClient

TOP_K_PER_QUERY = 5
ACTS_FILTER_MAP = {
    "BNS": "BNS",
    "BNSS": "BNSS",
    "BSA": "BSA"
}

async def qdrant_retriever_node(state: PipelineState) -> PipelineState:
    qdrant = QdrantClientWrapper()
    embedder = EmbeddingClient()

    all_results = []
    seen_sections = set()

    for query in state["expanded_queries"]:
        try:
            query_vector = await embedder.encode(query)

            # Filter by detected acts if specific acts identified
            filter_condition = None
            if state["detected_acts"] and len(state["detected_acts"]) < 3:
                filter_condition = {
                    "must": [{
                        "key": "act",
                        "match": {"any": state["detected_acts"]}
                    }]
                }

            results = await qdrant.search(
                collection_name="nyayasetu_clauses",
                query_vector=query_vector,
                limit=TOP_K_PER_QUERY,
                query_filter=filter_condition
            )

            for result in results:
                section_key = f"{result.payload['act']}-{result.payload['section_number']}"
                if section_key not in seen_sections:
                    seen_sections.add(section_key)
                    all_results.append({
                        "act": result.payload["act"],
                        "section_number": result.payload["section_number"],
                        "section_title": result.payload["section_title"],
                        "clause_text": result.payload["clause_text"],
                        "relevance_score": result.score,
                        "source": "qdrant"
                    })

        except Exception as e:
            state["errors"].append(f"qdrant_retriever query '{query[:30]}': {str(e)}")

    state["qdrant_results"] = sorted(
        all_results,
        key=lambda x: x["relevance_score"],
        reverse=True
    )[:15]  # Keep top 15 before merging

    return state
```

### 3.5 Node 3B — Neo4j Graph Traverser

```python
# pipeline/nodes/neo4j_traverser.py
from clients.neo4j_client import Neo4jClientWrapper

TRAVERSAL_DEPTH = 2
RELATIONSHIP_TYPES = [
    "REFERENCES",
    "PROCEDURAL_DEPENDENCY",
    "EVIDENCE_REQUIREMENT",
    "CROSS_ACT_LINK",
    "EXCEPTION_OF"
]

GRAPH_TRAVERSAL_QUERY = """
MATCH (c:Clause)
WHERE c.id IN $section_ids
CALL apoc.path.subgraphNodes(c, {
    relationshipFilter: $rel_filter,
    maxLevel: $depth
}) YIELD node
WHERE node:Clause AND node.id NOT IN $section_ids
RETURN DISTINCT
    node.id AS id,
    node.act AS act,
    node.section AS section_number,
    node.title AS section_title,
    node.text AS clause_text,
    0.75 AS relevance_score,
    'neo4j' AS source
LIMIT 10
"""

async def neo4j_traverser_node(state: PipelineState) -> PipelineState:
    neo4j = Neo4jClientWrapper()

    if not state.get("qdrant_results"):
        state["neo4j_results"] = []
        return state

    # Take section IDs from top Qdrant results as traversal seeds
    seed_ids = [
        f"{r['act']}-{r['section_number']}"
        for r in state["qdrant_results"][:5]
    ]

    rel_filter = "|".join(RELATIONSHIP_TYPES)

    try:
        records = await neo4j.run(
            GRAPH_TRAVERSAL_QUERY,
            section_ids=seed_ids,
            rel_filter=rel_filter,
            depth=TRAVERSAL_DEPTH
        )

        state["neo4j_results"] = [
            {
                "act": r["act"],
                "section_number": r["section_number"],
                "section_title": r["section_title"],
                "clause_text": r["clause_text"],
                "relevance_score": r["relevance_score"],
                "source": "neo4j"
            }
            for r in records
        ]

    except Exception as e:
        state["errors"].append(f"neo4j_traverser: {str(e)}")
        state["neo4j_results"] = []
        # Graceful degradation — continue without graph results

    return state
```

### 3.6 Node 4 — Result Merger and Deduplicator

```python
# pipeline/nodes/result_merger.py

def result_merger_node(state: PipelineState) -> PipelineState:
    qdrant_results = state.get("qdrant_results", [])
    neo4j_results = state.get("neo4j_results", [])

    seen = {}

    # Qdrant results take priority — higher confidence scores
    for clause in qdrant_results:
        key = f"{clause['act']}-{clause['section_number']}"
        seen[key] = clause

    # Neo4j results added if not already present
    for clause in neo4j_results:
        key = f"{clause['act']}-{clause['section_number']}"
        if key not in seen:
            seen[key] = clause

    merged = list(seen.values())

    # Re-rank: Qdrant source weighted 1.0x, Neo4j source 0.8x
    for clause in merged:
        if clause["source"] == "neo4j":
            clause["relevance_score"] = clause["relevance_score"] * 0.8

    merged_sorted = sorted(
        merged,
        key=lambda x: x["relevance_score"],
        reverse=True
    )[:10]  # Top 10 after merge

    # Assign final ranks
    for i, clause in enumerate(merged_sorted):
        clause["rank"] = i + 1

    state["merged_clauses"] = merged_sorted
    return state
```

### 3.7 Node 5 — Confidence Evaluator

```python
# pipeline/nodes/confidence_evaluator.py

HIGH_STAKES_KEYWORDS = [
    "murder", "rape", "kidnapping", "death", "suicide",
    "child", "minor", "acid attack", "terrorism",
    "hatyaa", "atmahatya", "bachcha", "maut"  # Hindi equivalents
]

CONFIDENCE_THRESHOLDS = {
    "escalate": 0.45,     # Below this → escalate
    "warn": 0.60,         # Below this → show warning to citizen
    "confident": 0.75     # Above this → full confidence
}

def confidence_evaluator_node(state: PipelineState) -> PipelineState:
    clauses = state.get("merged_clauses", [])
    high_stakes_raw = state.get("_high_stakes_raw", [])
    raw_input_lower = state["raw_input"].lower()

    # Calculate confidence score
    if not clauses:
        confidence = 0.0
    else:
        top_scores = [c["relevance_score"] for c in clauses[:3]]
        confidence = sum(top_scores) / len(top_scores)
        # Penalise if only 1 or 2 results found
        if len(clauses) < 3:
            confidence *= 0.7

    state["confidence_score"] = round(confidence, 3)

    # Detect high stakes
    high_stakes_flag = None
    if high_stakes_raw:
        high_stakes_flag = high_stakes_raw[0]
    else:
        for keyword in HIGH_STAKES_KEYWORDS:
            if keyword in raw_input_lower:
                high_stakes_flag = "high_stakes_keyword_detected"
                break

    state["high_stakes_flag"] = high_stakes_flag

    # Routing decision
    if confidence < CONFIDENCE_THRESHOLDS["escalate"] or high_stakes_flag:
        state["routing_decision"] = "escalate"
    else:
        state["routing_decision"] = "proceed"

    return state
```

### 3.8 Node 6 — Response Synthesiser

```python
# pipeline/nodes/response_synthesiser.py

SYNTHESIS_PROMPT = """
You are NyayaSetu, a legal intelligence system for Indian citizens.
You explain Indian law clearly and accurately.

CRITICAL RULES — these are absolute and cannot be broken:
1. You may ONLY reference clauses that appear in the RETRIEVED CLAUSES section below
2. You may NOT add, invent, or infer any clause not in the retrieved set
3. Every explanation must cite its source clause (act + section number)
4. Plain explanations must be in {language} — simple language a non-lawyer can understand
5. Relevance context must explain specifically why this clause applies to THIS situation

Citizen's situation:
<situation>{translated_input}</situation>

Retrieved clauses (you may ONLY use these):
<retrieved_clauses>
{retrieved_clauses_json}
</retrieved_clauses>

For each clause in the retrieved set, produce:
{{
  "section_number": "<from retrieved>",
  "act": "<from retrieved>",
  "section_title": "<from retrieved>",
  "clause_text": "<exact text from retrieved — do not paraphrase>",
  "plain_explanation": "<2–3 sentences in {language} explaining what this means>",
  "relevance_context": "<1–2 sentences explaining why this applies to the citizen's specific situation>",
  "relevance_score": <score from retrieved>
}}

Return a JSON array of these objects, ordered by relevance_score descending.
Do not add any text outside the JSON array.
"""

async def response_synthesiser_node(state: PipelineState) -> PipelineState:
    llm = LLMClient()

    retrieved_json = json.dumps(state["merged_clauses"], ensure_ascii=False)
    lang = state.get("detected_language", "en")
    output_language = "Hindi" if lang in ["hi", "hinglish"] else "English"

    prompt = SYNTHESIS_PROMPT.format(
        translated_input=state["translated_input"],
        retrieved_clauses_json=retrieved_json,
        language=output_language
    )

    try:
        response = await llm.complete(prompt, max_tokens=3000)
        synthesised = json.loads(response)

        # HALLUCINATION GUARD
        # Verify every returned section exists in merged_clauses
        valid_sections = {
            f"{c['act']}-{c['section_number']}"
            for c in state["merged_clauses"]
        }
        verified = []
        for clause in synthesised:
            key = f"{clause['act']}-{clause['section_number']}"
            if key in valid_sections:
                verified.append(clause)
            else:
                state["errors"].append(
                    f"HALLUCINATION BLOCKED: {key} not in retrieved set"
                )

        state["synthesised_clauses"] = verified

    except Exception as e:
        state["errors"].append(f"response_synthesiser: {str(e)}")
        # Fallback: return merged clauses without synthesis
        state["synthesised_clauses"] = state["merged_clauses"]

    return state
```

## 5. LLM Client Abstraction

```python
# clients/llm_client.py
from abc import ABC, abstractmethod
from config import settings

class BaseLLMClient(ABC):
    @abstractmethod
    async def complete(self, prompt: str, max_tokens: int) -> str:
        pass

class GroqClient(BaseLLMClient):
    def __init__(self):
        from groq import AsyncGroq
        self.client = AsyncGroq(api_key=settings.GROQ_API_KEY)
        self.model = "llama-3.3-70b-versatile"

    async def complete(self, prompt: str, max_tokens: int = 1000) -> str:
        response = await self.client.chat.completions.create(
            model=self.model,
            messages=[{"role": "user", "content": prompt}],
            max_tokens=max_tokens,
            temperature=0.1   # Low temperature for legal accuracy
        )
        return response.choices[0].message.content

class ClaudeClient(BaseLLMClient):
    def __init__(self):
        import anthropic
        self.client = anthropic.AsyncAnthropic(api_key=settings.ANTHROPIC_API_KEY)
        self.model = "claude-sonnet-4-6"

    async def complete(self, prompt: str, max_tokens: int = 1000) -> str:
        response = await self.client.messages.create(
            model=self.model,
            max_tokens=max_tokens,
            messages=[{"role": "user", "content": prompt}],
            temperature=0.1
        )
        return response.content[0].text

def LLMClient() -> BaseLLMClient:
    """Factory — swap provider via config"""
    if settings.LLM_PROVIDER == "groq":
        return GroqClient()
    elif settings.LLM_PROVIDER == "claude":
        return ClaudeClient()
    else:
        raise ValueError(f"Unknown LLM provider: {settings.LLM_PROVIDER}")
```

## 7. Error Handling Strategy

|Error Type|Handler|User Response|
|---|---|---|
|LLM timeout (>30s)|Retry × 3 with exponential backoff (2s, 4s, 8s)|"Processing is taking longer than usual. Please wait."|
|LLM rate limit (429)|Queue with 60s delay, retry|"High demand detected. Your query is queued."|
|Qdrant unavailable|Return graceful degradation — skip vector search|"Partial results may be returned."|
|Neo4j unavailable|Skip graph traversal, proceed with Qdrant only|Silent degradation — citizen sees no difference|
|Hallucination blocked|Log blocked clause, remove from output|Output returned without the hallucinated clause|
|Payment verification failed|Do not generate writeup, return clear error|"Payment could not be verified. Please try again or contact support."|
|PDF generation failed|Retry once, then flag for manual review|"Your document is being prepared. Download link will be emailed to you."|
|Pipeline total failure|Log full trace, return 500 with request ID|"Something went wrong. Your query has been saved (ID: xxx). We will process it shortly."|

## 9. Design Patterns Used

|Pattern|Where Used|Why|
|---|---|---|
|Factory|`LLMClient()` function|LLM provider swappable via config with zero code change|
|Strategy|Each LangGraph node|Each node is independently replaceable without changing pipeline structure|
|Repository|Service layer → SQLAlchemy models|Business logic decoupled from database implementation|
|Dependency Injection|FastAPI `Depends()`|Auth, DB sessions, and rate limiter injected — not imported globally|
|Circuit Breaker|Neo4j client|If Neo4j fails 3× in 60s, skip for 5 minutes — prevents cascade failure|
|Graceful Degradation|Neo4j traversal node|System returns valid output even if graph layer is unavailable|
